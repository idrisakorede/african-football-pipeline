# African Football Data Pipeline

A multi-phase data engineering project scraping football league data across Africa, evolving from local pipeline to lakehouse architecture.

## Phases
- **Phase 1** (current): Local scraper with sructured file output
- **Phase 2**: Warehouse-centric - S3, Airflow, Snowflake, dbt
- **Phase 3**: Lakehouse - Databricks, Delta Lake, architecture comparison

## Data Source
Soccerway - covering leagues across the African continent.
